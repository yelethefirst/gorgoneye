Phishing protection that doesn't read your email.

Aegis Gorgon is a privacy-preserving browser extension that catches phishing locally — at the moment you hover a link. Email content never leaves your device. Full URLs are never sent to any service. Every outbound byte is audited.

How it works

Five independent layers. One verdict. Full explanation.

Each layer runs on its own and contributes evidence. The fusion engine combines them via a transparent weighted noisy-OR; you see the breakdown in the popup.

1

Rules engine

Nine deterministic checks for typosquatting, homographs, embedded credentials, URL-in-URL redirects, and more — sub-millisecond, fully local.

2

Local ML classifier

On-device XGBoost model exported to ONNX. 25 features, runs in onnxruntime-web, never sees the network.

3

Threat intelligence

Google Safe Browsing via the privacy-preserving 4-byte hash-prefix flow — the URL itself is never transmitted.

4

Visual brand matching

Perceptual-hash check against a local brand database. Catches lookalike login pages on look-clean domains. Consent-gated.

5

Explanation

Verdicts come with a 2–3 sentence summary plus actionable guidance. Optional local LLM; deterministic template fallback always available.

★

Inline + popup UI

Badges render beside Gmail / Outlook links. Hover anywhere else for a mini warning. Click the badge for the full layer breakdown.

Verdicts

Safe No risk signals fired. Score < 0.30.
Suspicious One or more risk signals fired. Confidence between 0.30 and 0.74.
Phishing Multiple independent strong signals, or a confirmed Safe Browsing match. Confidence ≥ 0.75.

Privacy

The promises, in writing.

These aren't marketing copy. Every promise here is enforced structurally in code and verified by the one-click privacy verifier on the options page.

✓ Email body content never leaves your device.
✓ Full scanned URLs are never sent to Aegis-controlled services.
✓ Safe Browsing lookups use the 4-byte hash-prefix flow only — never the URL.
✓ Every outbound network call goes through an audited fetch wrapper.
✓ The audit log is visible on the options page and exports as JSON.
✓ Telemetry is opt-in. There is no telemetry implementation today.
✓ Visual inspection contacts the target origin only with explicit per-scan consent.
✓ The privacy verifier replays the full pipeline locally and shows pass/fail checks.

Install

Two-minute setup.

  1. Get the build. Clone the repo and run pnpm install, then pnpm build. The extension lands in .output/chrome-mv3/.
  2. Load it unpacked. Open chrome://extensions, turn on Developer Mode, click Load unpacked, and pick the build directory.
  3. Pin the icon. Click the puzzle-piece, find Aegis Gorgon, hit the pin.
  4. Open the welcome tour. It opens automatically on first install; the privacy posture and what stays local are explained in two screens.

Firefox MV2 build: pnpm build:firefox. Load .output/firefox-mv2/manifest.json via about:debugging.
